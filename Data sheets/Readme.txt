This directory contains key data sheets for the components used in our project:

1)Components data sheet (Word document)
Contains data sheets for commonly used components in HSDSL Lab projects, such as transistors, capacitors, and other standard electronic parts.

2)GM810 Series Barcode reader module (HTML file)
Includes the data sheet for the barcode reader module used in our project.

3)usb_charger_tps63020 (HTML file)
Provides the data sheet for the USB Type-C charger component integrated into the project.


These files are useful for understanding the specifications and characteristics of the components utilized in the design.
